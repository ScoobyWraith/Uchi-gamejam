# Проект для GameJam VK Education x Учи.ру

## Идея

Игровой проект, цель которого мотивировать пользователя проводить время вне экрана вычислительного устройства, занимаясь творческой или физической деятельностью.

В текущей реализации игры представлен один вид активности: **оригами**, то есть сборка фигурок из бумаги.

Данная активность подается как часть сюжета, в рамках которого пользователь занимается сборкой персонажей игры, которые помогают игроку достичь финальной цели - найти капитана и передать ему карту сокровищ.

Активность подается в виде викторины, правильный ответ на которую можно дать, собрав очередного персонажа из бумаги.

В качестве награды и мотивации продолжать игру, пользователю предоставляется экранная активность в виде одной из двух игр: бег с препятсвиями или бег с уклонением.

Иные виды активностей, которые не были реализованы в силу ограничения времени:  
* **картина по номерам**: пользователю предлагается закрашивать клетки на клетчатом поле согласно числовому значению, характеризующему определенный цвет. По завершению процесса закрашивания следует вопрос (с вариантами ответов), какой персонаж получился;
* **наборе текстовых инструкций**: пользователю предлагается совершать простые действия на клетчатом поле, по итогу которых задается вопрос (с вариантами ответов), какая геометрическая фигура получилась или вопрос, в котором нужно определить какому из представленных рисунков наиболее близок результат;
* **смешивание цветов**: пользователю предлагается воспользоваться красками и смешать два и более заданных цветов с целью определения получившегося цвета в результате. Данный цвет служит для раскрашивания очередного персонажа по сюжету.

## Стек технологий

* Unity
* FMOD Studio
* C#

## Компоненты системы
Текущий прототип не имеет сложных технических компонентов в виде backend части. Сохранение прогресса игры выполнено с помощью PlayerPrefs. Игра дает возможность начать игру с начала (опционально) при очередной перезагрузке игры.

## Запуск
Для запуска игры достаточно браузера, в котором открывается WebGL версия игры.

Адрес странички с игрой: https://scoobywraith.github.io/Uchi-gamejam/

Целевое разрешение игры: 1200 x 675.
